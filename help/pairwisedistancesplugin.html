Pairwise Distances Plugin

Pairwise Distances Plugin is an extension of Gama-q2 for excluding outlying coupples of distances based on Normal distribution test.

Quick Start

Select or deselect some measurements, which should not be used in adjustement. Outlaying coupples are marked by red text-color.

File menu

This menu contains some basic functions.

Close

Close this plugin.

Settings menu

It allows to you customize settings here.

Reset

Data are cleared from the table and reloaded from Gama-q2. Precission were reseted, too.

Set All Active

It sets all measurements (angles and directions) in local network active.

Set All Passive

It sets all measurements (angles and directions) in local network passive.

Sort By Quality Asc

Sort by ratio of normal distribution test and absolute value of closer in ascending order.

Sort By Quality Desc

Sort by ratio of normal distribution test and absolute value of closer in descending order.

Distances

Row filter for excluding or including horizontal distances.

Slope Distances

Row filter for excluding or including slope distances.

Unused

Row filter for excluding or including used or unused measurements.

Type of Measurements

Column filter for showing or hidding type of measurements.

Orientation of Distances

Column filter for showing or hidding orientation of distances.

Standard Deviation

Column filter for showing or hidding standard deviations.

Normal Distribution Test

Column filter for showing or hidding Normal Distribution Test.

Settings

It shows a dialog which allows to you manage an output precission.
